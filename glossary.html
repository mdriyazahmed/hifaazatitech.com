Cybersecurity Glossary

Comprehensive dictionary of cybersecurity terms and definitions

Advanced Persistent Threat (APT)
A prolonged and targeted cyberattack where an intruder gains access to a network and remains undetected for an extended period while stealing data.
Authentication
The process of verifying the identity of a user, device, or system before granting access to resources.
Botnet
A network of compromised computers controlled remotely by cybercriminals to perform malicious activities.
Chief Information Security Officer (CISO)
Senior-level executive responsible for developing and implementing an organization's information security strategy.
Distributed Denial of Service (DDoS)
A cyberattack that attempts to make a service unavailable by overwhelming it with traffic from multiple sources.
Encryption
The process of converting readable data into coded form to prevent unauthorized access during transmission or storage.
Firewall
A network security device that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
Honeypot
A security mechanism that creates a trap to detect, deflect, or study attempts at unauthorized network access.
Incident Response
The organized approach to addressing and managing security breaches or cyberattacks, including preparation, detection, and recovery.
Malware
Malicious software designed to damage, disrupt, or gain unauthorized access to computer systems, including viruses, worms, and ransomware.
Penetration Testing
A simulated cyberattack against a computer system to check for exploitable vulnerabilities and assess security posture.
Phishing
A cybercrime where attackers impersonate legitimate entities to steal sensitive information like usernames, passwords, and credit card details.
Security Information and Event Management (SIEM)
A software solution that aggregates and analyzes activity from many different resources across an IT infrastructure.
Security Operations Center (SOC)
A centralized unit that deals with security issues on an organizational and technical level, monitoring and analyzing security events.
Vulnerability
A weakness in a system that can be exploited by threats to gain unauthorized access or perform unauthorized actions.
Zero-day Attack
A cyberattack that occurs on the same day a weakness is discovered in software, before a security patch is available.
