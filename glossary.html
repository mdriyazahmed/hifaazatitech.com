Cybersecurity Glossary

Essential cybersecurity terminology, definitions, and concepts

Common Threats

Malware

Malicious software designed to harm, exploit, or otherwise compromise a computer system or network.

Phishing

Fraudulent attempt to obtain sensitive information by disguising as a trustworthy entity in electronic communication.

Ransomware

Type of malware that encrypts files and demands payment for decryption key.

Zero-Day Exploit

Attack that targets a previously unknown vulnerability in software before a patch is available.

Network Security

Firewall

Network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

Intrusion Detection System (IDS)

Device or software that monitors network traffic for suspicious activity and sends alerts when detected.

VPN (Virtual Private Network)

Encrypted connection over a less secure network, typically the internet, providing privacy and anonymity.

DDoS (Distributed Denial of Service)

Attack where multiple compromised systems target a single system causing denial of service for users.

Authentication & Access Control

Multi-Factor Authentication (MFA)

Security system requiring multiple methods of authentication from independent categories of credentials.

Single Sign-On (SSO)

Authentication scheme allowing users to log in with a single ID to any of several related software systems.

Access Control List (ACL)

List of permissions attached to an object specifying which users or system processes can access objects.

Least Privilege

Security principle in which users are given minimum levels of access necessary to complete job functions.

Data Protection

Data Breach

Security incident in which sensitive, protected, or confidential data is accessed or disclosed without authorization.

Data Loss Prevention (DLP)

Strategy and tools used to ensure that sensitive data is not lost, misused, or accessed by unauthorized users.

End-to-End Encryption (E2EE)

Communication system where only the communicating users can read the messages, preventing third-party access.

Hashing

Process of converting data of any size into a fixed-size string of characters, typically for security purposes.

Compliance & Governance

GDPR

General Data Protection Regulation - European Union regulation on data protection and privacy.

HIPAA

Health Insurance Portability and Accountability Act - US legislation providing data privacy provisions for safeguarding medical information.

ISO 27001

International standard for information security management systems (ISMS).

PCI DSS

Payment Card Industry Data Security Standard - set of security standards designed to ensure companies process, store, and transmit credit card information securely.
