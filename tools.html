Cybersecurity Tools

Curated utilities for assessment, monitoring, response, and hardening—updated regularly.

Assessment

Nmap Network

Network discovery and security auditing. Profiles open ports and services.

Command: nmap -sC -sV -Pn target

OWASP ZAP Web

Intercepting proxy and scanner for web application security testing.

Use: Passive scan during exploratory testing; CI baseline scans.

Trivy Containers

Scanner for container images, file systems, and IaC misconfigurations.

CI: trivy image --severity HIGH,CRITICAL repo/image:tag

Monitoring

Wazuh XDR/SIEM

Open-source security platform for endpoint monitoring and incident detection.

Tip: Centralize agent logs and enable FIM + vulnerability DB sync.

Prometheus + Grafana Metrics

Time-series metrics and visualization for infra and app SLOs.

Alerting: Define thresholds for auth failures and error spikes.

Zeek Network

Network security monitoring framework generating rich protocol logs.

Parse: Ship logs to SIEM for correlation and detections.

Response

Velociraptor DFIR

Endpoint visibility and collection through VQL. Ideal for threat hunting.

Playbook: Snapshot triage: processes, persistence, netconns, autoruns.

Cyber Triage IR

Guided triage of compromised hosts with scoring of suspicious artifacts.

Export: Generate case bundle for chain-of-custody.

YARA Detection

Pattern matching for malware and IOC hunting across file systems/memory.

Rule: use meta + strings + condition; test on benign samples.

Hardening

OpenSCAP Compliance

Audit and remediate systems against CIS/SCAP benchmarks.

CLI: oscap xccdf eval --profile xccdf_org.ssgproject.content_profile_cis

tfsec IaC

Static analysis for Terraform to catch insecure defaults pre-deploy.

CI: tfsec . --soft-fail for gradual adoption.

Dependabot Supply Chain

Automated dependency updates with security advisories in pull requests.

Tip: Require passing security tests before merge.
